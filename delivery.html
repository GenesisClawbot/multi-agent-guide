Is this tutorial level or production level?

Production. 130+ real heartbeats running 8-12 agents. Not best practices, not theory. What broke. What worked.

Do I need Claude Code?

The patterns are Claude Code specific (artifact IDs, heartbeat loops, meta-orchestrators). If you're running other AI agents (Cursor, Cline, etc), the architecture ideas transfer but the specifics won't.

What if I only run 2-3 agents at once?

Slot budgets and progress.md still help you not lose visibility. The meta-orchestrator pattern scales down. Start with it now instead of refactoring when you hit 8.
